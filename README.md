# Guido Celada Website

Sitio personal estático para `guidocelada.com`, creado con HTML, CSS y un pequeño efecto canvas en JavaScript.

## Deploy en GitHub Pages

1. Subir este repositorio a GitHub.
2. En GitHub, abrir `Settings > Pages`.
3. Configurar el source como branch `main` y repository root.
4. Apuntar el dominio a GitHub Pages y mantener el archivo `CNAME` en la raíz.

## Editar redes

Los links de redes están en `index.html`. Instagram y TikTok usan `@ia.con.guido`.
